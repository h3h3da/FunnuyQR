# FunnuyQR
基于[qrcode](https://github.com/h3h3da/qrcode) 开发的本地二维码生成小应用。

## 作用

在本地生成二维码，可以加背景，可选择黑白或彩色二维码。

## 环境

### 1.Windows

### 2.Python 3

### 3.pip 3

### [qrcode](https://github.com/h3h3da/qrcode)

tips:具体可参考[qrcode](https://github.com/h3h3da/qrcode) 的安装要求。
